# Spark Editor - Visual Studio Extension

Visual configuration editor for MintPlayer.Spark applications.

## Development Setup

1. Set **SparkEditor.Vsix** as the startup project.
2. Open **Build > Configuration Manager** and check the **Deploy** checkbox for the SparkEditor.Vsix project. This is unchecked by default and the extension will not be installed in the experimental instance without it.
3. Press F5 to launch the Visual Studio experimental instance with the extension loaded.

The menu item appears under **View > Spark Editor**.
