MintPlayer Spark - JSON Schemas

JSON Schema definitions for MintPlayer Spark application configuration files. Add IntelliSense and validation to your App_Data JSON files.

Available Schemas (v1)

Entity Type

Defines entity types (persistent objects) with attributes, tabs, groups, validation rules, and associated queries. Used in App_Data/Model/*.json files.

https://json.spark.mintplayer.com/schemas/v1/entity-type.schema.json

Program Units

Defines the navigation menu structure with groups and program units linking to queries or persistent objects. Used in App_Data/programUnits.json.

https://json.spark.mintplayer.com/schemas/v1/program-units.schema.json

Translations

Defines UI translation strings as key-value pairs where each value is a translated string with language codes. Used in App_Data/translations.json.

https://json.spark.mintplayer.com/schemas/v1/translations.schema.json

Culture Configuration

Defines supported languages with their translated display names and the default language. Used in App_Data/culture.json.

https://json.spark.mintplayer.com/schemas/v1/culture.schema.json

Security Configuration

Defines security groups and permission rights for the authorization system. Used in App_Data/security.json with the MintPlayer.Spark.Authorization package.

https://json.spark.mintplayer.com/schemas/v1/security.schema.json

Custom Actions

Defines custom actions that can be executed on entity instances, with display metadata, confirmation dialogs, and selection rules. Used in App_Data/customActions.json.

https://json.spark.mintplayer.com/schemas/v1/custom-actions.schema.json

Usage

Add a $schema property at the top of your JSON files to enable IntelliSense and validation in your editor (VS Code, Visual Studio, JetBrains IDEs):

Entity Type (App_Data/Model/Car.json)

{
  "$schema": "https://json.spark.mintplayer.com/schemas/v1/entity-type.schema.json",
  "persistentObject": {
    "id": "facb6829-f2a1-4ae2-a046-6ba506e8c0ce",
    "name": "Car",
    "clrType": "Fleet.Entities.Car",
    "attributes": []
  },
  "queries": []
}

Program Units (App_Data/programUnits.json)

{
  "$schema": "https://json.spark.mintplayer.com/schemas/v1/program-units.schema.json",
  "programUnitGroups": []
}

Translations (App_Data/translations.json)

{
  "$schema": "https://json.spark.mintplayer.com/schemas/v1/translations.schema.json",
  "save": { "en": "Save", "fr": "Enregistrer", "nl": "Opslaan" },
  "cancel": { "en": "Cancel", "fr": "Annuler", "nl": "Annuleren" }
}

Culture (App_Data/culture.json)

{
  "$schema": "https://json.spark.mintplayer.com/schemas/v1/culture.schema.json",
  "languages": {
    "en": { "en": "English", "fr": "Anglais", "nl": "Engels" }
  },
  "defaultLanguage": "en"
}

Security (App_Data/security.json)

{
  "$schema": "https://json.spark.mintplayer.com/schemas/v1/security.schema.json",
  "groups": {
    "00000000-0000-0000-0000-000000000000": { "en": "Everyone" }
  },
  "rights": []
}

Custom Actions (App_Data/customActions.json)

{
  "$schema": "https://json.spark.mintplayer.com/schemas/v1/custom-actions.schema.json",
  "CarCopy": {
    "displayName": { "en": "Copy Car" },
    "icon": "Copy",
    "showedOn": "both",
    "selectionRule": "=1",
    "refreshOnCompleted": true
  }
}
